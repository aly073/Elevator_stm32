# Elevator STM32

## Description

A fully functional elevator system based on STM32 (STM32F103C8T6 - The Blue Pill). A 100% of the code is written in ARM Assembly.

## Features

- RFID access
- Bluetooth mobile authintication
- Maximum weight detection
- Floor animations on led matrix screen
- SCAN scheduling algorithm with middle-floor direction logic
- Speaker feedback (auth sounds, du’aa on startup)
- Automatic doors with line-cut detection
- Full startup calibration routine
